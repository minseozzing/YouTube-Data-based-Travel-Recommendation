# S14P21D206 프로젝트 공유 문서 (기능명세 기준)

## 1. 프로젝트 한 줄 정의
예산/기간/위험민감도와 사용자 취향(유튜브 연동 또는 수동 선택)을 기반으로, 여행지를 추천하고 물가/항공권/뉴스 근거를 함께 제공하는 여행 의사결정 서비스.

## 2. 이 문서의 기준
- 본 문서는 `기획서` 아이디어보다 `기능명세서.csv`, `기능명세서2.csv`를 우선 기준으로 작성함
- 즉, 실제 개발 범위와 화면 구성은 기능명세의 PAGE 코드 기준으로 이해

## 3. 핵심 사용자 흐름
1. 로그인 (Google)
2. 취향 입력/연동 선택
3. 메인에서 예산/기간/위험민감도 입력
4. 추천 결과 확인 (리스트 + 지구본)
5. 지역 상세 모달에서 물가/항공권/뉴스 확인
6. 저장(북마크/리포트) 후 마이페이지에서 재조회

## 4. 기능명세 기준 화면/기능

### PAGE-101 로그인 페이지
- Google 소셜 로그인
- 로그인 성공 시 메인 이동

### PAGE-102 취향 선택 페이지
- 유튜브 연동 불가/미동의 시 수동 취향 선택
- 예: 자연, 휴양, 문화/역사, 액티비티, 맛집, 가성비

### PAGE-201 메인 페이지
- 입력: 예산, 기간, 위험민감도
- 출력: 추천 여행지 목록(리스트 카드), 지구본 간략 정보

### PAGE-202 메인 페이지(지구본)
- 레이어: 물가, 항공 비용, 안전 등급
- 표시 기준:
  - 물가: 원화 환산
  - 항공 비용: 인천공항 기준
  - 위험도: 상/중/하

### PAGE-203 메인 페이지(유튜브 연동)
- Google 동의 페이지 리다이렉트
- 동의 여부 상태 표시

### PAGE-205 상세 정보 모달
- 지구본/리스트에서 지역 선택 시 모달 오픈
- 탭: 물가 비교, 항공권, 뉴스

### PAGE-205-1 상세 모달 저장
- 조회 시점의 지역 정보를 저장
- 저장 항목: 물가 요약, 취향, 당시 위험도

### PAGE-206 도시별 추천 탭
- 추천 이유 설명 제공
- 항목: 물가, 취향 매칭, 위험도

### PAGE-207 / PAGE-207-1 생활물가 비교
- 한국 대비 국가/도시 물가 비교
- PPP 지수 추이 그래프
- 항목별 비교(식사, 교통비 등)

### PAGE-208 항공권 탭
- 월별 최저가 그래프
- 최저가 날짜
- 비행시간
- 경유 횟수

### PAGE-210 뉴스 탭
- 선택 도시/국가 관련 뉴스 요약
- 뉴스 리스트(제목, URL, 가능하면 이미지)

### PAGE-301 마이페이지
- 회원 탈퇴(소프트 딜리트)
- 선호도/성향 수정
- 저장된 여행지 리포트 조회

### PAGE-302 물가 페이지(검색)
- 필터: 대륙, 국가, 도시(검색어)
- 결과 분기:
  - 대륙 검색: 국가 물가 카드 리스트
  - 국가/도시 검색: 물가 상세 페이지

### PAGE-401 항공권 (부가)
- 크롤링 수준에 따라 월/일 단위 상세 항공 정보 제공
- 예: 가격 추이, 달력, 일별 가격, 숙박비, 경유 횟수, 비행 시간

## 5. MVP/부가 기능 구분

### MVP(우선 구현 권장)
- PAGE-101, 102, 201, 202, 203
- PAGE-205, 206
- PAGE-207, 208, 210
- PAGE-301(기본 조회/수정/탈퇴)

### 부가/확장
- PAGE-401 항공권 고도화
- 북마크 리포트 고도화(리포트 포맷 강화)

## 6. 팀 공통 합의 포인트
- 추천 결과는 "왜 추천했는지"를 반드시 설명 가능해야 함
- 유튜브 연동 미동의 사용자를 위한 대체 입력 경로(수동 취향 선택)는 필수
- 단순 추천 리스트보다, 물가/항공권/뉴스 근거를 함께 제공하는 UX가 핵심
