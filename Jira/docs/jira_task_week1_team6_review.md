# 1주차 Jira 테스트 이슈 초안 (팀원 6명 / 검토용)

## 반영 기준
- 기준 문서: `Jira/docs/jira_task.md`의 `1주차 – 설계 + 기반 구축`
- 1주차 목표:
  - 전체 구조 확정
  - 데이터 수집이 실제로 돌아가게 만들기
  - API 뼈대 완성
  - 화면 골격 완성
- Jira 컨벤션(`jira.md`):
  - Epic: 도메인명
  - 제목 Prefix: `Feat:`, `Docs:`, `Style:`
  - 설명 첫 줄: `[Role] 사용자는 {목적}을 하기 위해 {기능명}을 사용하고 싶다.`
  - Role: `FE`, `BE`, `INF`, `BD`

## 작업 전 상태
- 기존 생성분 `S14P21D206-35`~`S14P21D206-55` 삭제 완료
- 이번 문서는 **백로그 등록 전 검토용** (아직 미생성)
