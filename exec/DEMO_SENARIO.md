# DEMO SCENARIO

## 1. 시연 목표
- Google/YouTube 취향 분석 기반 여행지 추천
- 추천 도시 상세 정보(이유, 물가, 항공권, 관광지) 탐색
- 북마크 저장 및 도시 물가 비교 기능 시연
