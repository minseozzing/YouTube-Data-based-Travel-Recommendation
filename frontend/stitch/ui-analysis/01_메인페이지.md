# UI Structure Analysis: 메인페이지

## 1. Layout Pattern
두 컬럼 구조: 좌측 고정 사이드바(Trip Settings + Top Matching 목록) + 우측 유동 메인 콘텐츠 영역(히어로 텍스트 + 글로브 + 하단 통계 바). 전체 너비 상단 네비게이션 바. 모달/오버레이 없음. 전체 페이지 레이아웃.

## 2. Estimated Grid Structure
12컬럼 기반 추정. 좌측 사이드바 약 2–3컬럼, 메인 콘텐츠 9–10컬럼. 상단 네비게이션은 전체 12컬럼 점유. 하단 통계 바는 메인 콘텐츠 내 중앙 정렬되어 외곽 그리드 경계와 불일치(grid break). 사이드바-메인 경계에 명시적 거터 없음(고정폭 사이드바 + 유동 우측 구조).

## 3. Visual Hierarchy Order
- Tier 1: 히어로 헤드라인 ("Explore your next escape.") — 최대 폰트 크기, 고대비, cyan/teal 색상 강조어
- Tier 2: 글로브/지도 시각화 — 대형 원형 요소, 메인 콘텐츠 중앙 점유
- Tier 3: 서브 헤드라인/설명 텍스트 — 히어로 하단, 중간 대비
- Tier 4: 좌측 패널 Trip Settings 폼 컨트롤 (슬라이더, 인풋, CTA 버튼)
- Tier 5: Top Matching 목록 아이템 (시드니, 리스본) — 썸네일 + 배지
- Tier 6: 하단 통계 바 (탐험자 수, 항공비, 생활비, 안전등급)
- Tier 7: 상단 네비게이션 바 (로고, 검색, 링크, Join Premium 버튼, 아바타)
- Tier 8: 푸터 텍스트 (Privacy, Safety, 저작권)

## 5. Interaction Elements
| Element | Type | Likely Trigger |
|---|---|---|
| Budget 슬라이더 | range input | 예산 값 업데이트 |
| Duration 인풋 | number input | 기간 값 업데이트 |
| Risk Sensitivity 슬라이더 | range input | 위험도 필터 업데이트 |
| Update Recommendations 버튼 | button | 추천 결과 재조회 API 호출 |
| 글로브 위 핀 마커 | clickable marker | 도시 상세 모달 오픈 |

| Join Premium 버튼 | button | 프리미엄 가입 페이지 이동 |
| 검색 필드 | text input | 도시/국가 검색 |

## 6. Data Display Patterns
| Area | Display Type | Dynamic Data |
|---|---|---|
| 글로브 시각화 | map/globe | Yes |
| Top Matching 목록 | card-list | Yes |
| 하단 통계 바 | stat-number | Yes |
| Trip Settings 슬라이더 | input (range) | No |

## 7. Repeated Patterns
Top Matching 카드: 썸네일 이미지 + 도시명/국가명 + 매칭% 배지 + 예상 비용으로 구성된 카드 단위가 2회 반복. 확장 시 N개 반복 가능한 리스트 구조.

## 8. Dominant Element & Information Density
- **Dominant**: 히어로 헤드라인 ("Explore your next escape.") — 최대 폰트 크기 + cyan/teal 색상 강조 + 상단 중앙 우측 위치
- **Density**: Medium — 메인 콘텐츠 영역은 글로브 중심으로 여백이 넉넉하나(Low), 좌측 사이드바는 복수의 인터랙션 요소가 밀집(High). 전체 평균 Medium.
