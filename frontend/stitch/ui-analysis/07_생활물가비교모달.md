# UI Structure Analysis: 생활물가 비교 모달

## 1. Layout Pattern
modal-overlay. 배경 페이지 위 중앙 모달. 모달 내부: 비대칭 2열 (좌: 목적지 이미지 카드 약 30–35% / 우: 탭+콘텐츠 약 65–70%). 활성 탭: "생활물가 비교".

## 2. Estimated Grid Structure
모달 내부: 명시적 12컬럼 없음. 좌우 비대칭 2-zone 분할. 우측 콘텐츠 내부: 상단 2열 서브그리드 (스탯 카드 2개 나란히) + 하단 단일 컬럼 비교 테이블. 거터 일관성 약 16–20px 추정.

## 3. Visual Hierarchy Order
- Tier 1: 좌측 목적지 히어로 이미지 (시드니 카드) — 전체 높이, 최대 면적, 고채도 사진
- Tier 2: "시드니" 대형 흰색 볼드 텍스트 (이미지 오버레이) — 최대 폰트, 최고 대비
- Tier 3: "생활물가 비교" 활성 탭 — 언더라인/활성 상태 표시
- Tier 4: "약 15% 높을 수" 스탯 블록 — 볼드 수치 + "UPDATED TODAY" 배지
- Tier 5: "185,000 KRW" 월세 카드 — 대형 숫자
- Tier 6: PPP 지수 추이 바 차트 + 환율 그룹 바 차트 (나란히)
- Tier 7: 항목별 비교 테이블 행 (교통비, 식비, 주거비, 오락비)
- Tier 8: 상단 네비게이션 바 + 푸터 (배경)

## 7. Repeated Patterns
비교 테이블 행: 카테고리명 + 좌측(서울) 값 + 우측(비교 도시) 값 구조 4회 반복. 탭 아이템: 4회 반복.

## 8. Dominant Element & Information Density
- **Dominant**: 좌측 히어로 사진 카드 — 전체 컬럼 높이 점유, 최대 면적, 고채도 사진
- **Density**: Medium — 우측에 스탯 2개 + 차트 2개 + 테이블이 있으나 카드 경계와 여백으로 구분됨
