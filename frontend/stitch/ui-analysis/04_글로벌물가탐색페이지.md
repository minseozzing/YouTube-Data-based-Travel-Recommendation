# UI Structure Analysis: 글로벌 물가탐색 페이지

## 1. Layout Pattern
단일 컬럼 수직 스택 (full-page scroll). 전체 너비 네비게이션 + 히어로 섹션 + 카드 그리드 섹션 반복 + CTA 배너 + 푸터. 탭/모달 없음.

## 2. Estimated Grid Structure
12컬럼 기반 추정. 히어로 섹션은 1컬럼 중앙 정렬. "한국인이 자주 찾는 여행지 TOP" 행: 3-column 균등 카드 그리드 (거터 약 16–20px). "물가가 가장 저렴한 여행" 행: 4-column 균등 카드 그리드. "물가가 비싼 여행" 행: 4-column 균등 카드 그리드. CTA 배너: 전체 너비. 푸터: 약 4-column. 그리드 이탈 없음.

## 3. Visual Hierarchy Order
- Tier 1: 페이지 헤딩 "어디로 떠나볼까요?" — 최대 폰트 크기, 히어로 영역 중앙, 고대비
- Tier 2: "한국인이 자주 찾는 여행지 TOP" 섹션 헤더
- Tier 3: TOP 행 대형 사진 카드 3개 (full-bleed 이미지 + 오버레이 텍스트 + BEST 배지)
- Tier 4: "물가가 가장 저렴한 여행" / "물가가 비싼 여행" 섹션 헤더 (동등 가중치)
- Tier 5: 4-column 소형 카드들 (썸네일 + 국가명 + 가격)
- Tier 6: 전체 너비 파란 CTA 배너
- Tier 7: CTA 배너 내 본문 텍스트 + 버튼
- Tier 8: 푸터

## 5. Interaction Elements
| Element | Type | Likely Trigger |
|---|---|---|
| 히어로 검색 바/입력 | text input | 도시/국가 검색 |
| 대형 카드 (TOP 3) | clickable card | 국가/도시 상세 페이지 이동 |
| 소형 카드 (4열) | clickable card | 국가/도시 상세 페이지 이동 |
| "더보기" 링크 | link | 전체 목록 페이지 이동 |
| CTA 배너 버튼 | button | 추천/로그인 페이지 이동 |

## 6. Data Display Patterns
| Area | Display Type | Dynamic Data |
|---|---|---|
| TOP 여행지 카드 (3열) | image-grid | Yes |
| 저렴한 여행지 카드 (4열) | card-list | Yes |
| 비싼 여행지 카드 (4열) | card-list | Yes |
| 히어로 검색 | input | No |

## 7. Repeated Patterns
소형 카드: 썸네일 이미지 + 국가명 + 가격 구조가 4-column 행에서 4회 반복. 동일 패턴이 "저렴한 여행" / "비싼 여행" 두 섹션에 걸쳐 총 8회. 섹션 헤더 + 더보기 링크 + 카드 그리드의 섹션 단위 반복 패턴도 2회 반복.

## 8. Dominant Element & Information Density
- **Dominant**: 히어로 헤딩 "어디로 떠나볼까요?" — 최대 폰트 크기 + 중앙 정렬 + 고대비 + 최상단 콘텐츠 위치
- **Density**: Medium — 섹션 간 여백 확보, 카드 거터 균등. 단 다수 섹션 수직 적층으로 전체 페이지 길이는 높음.
