# UI Structure Analysis: 현지 뉴스 정보 모달

## 1. Layout Pattern
modal-overlay. 배경 페이지 위 중앙 모달. 모달 내부: 비대칭 2열 (좌: 목적지 카드 약 35% / 우: 탭+콘텐츠 약 65%). 활성 탭: "현지 뉴스". 우측 콘텐츠: 수직 구분선으로 좌(뉴스 요약)와 우(기사 스크롤) 2개 서브 컬럼 분리.

## 2. Estimated Grid Structure
모달 내부: 좌우 비대칭 2-zone 분할. 우측 콘텐츠: 암묵적 2-column 서브그리드 (약 55:45, 수직 구분선 포함). 각 서브 컬럼 내부는 단일 컬럼 수직 스택. 간격 기준 8–16px 추정.

## 3. Visual Hierarchy Order
- Tier 1: "시드니" 목적지 헤딩 — 좌측 카드 하단, 최대 폰트, 흰색 고대비 + 드롭 쉐도우
- Tier 2: "현지 뉴스" 활성 탭 — 활성 상태 표시
- Tier 3: "최신 뉴스 요약" 섹션 헤더 + 초록 배지 ("GOOD NEWS") — 고채도 색상 대비
- Tier 4: 뉴스 요약 불릿 항목 3개 (안전:, 날씨:, 교통: 볼드 리드인)
- Tier 5: "주요 기사 스크롤" 섹션 헤더 + "더보기 바로가기" 링크
- Tier 6: 기사 카드 목록 3개 (썸네일 + 제목 + 출처 + 날짜)
- Tier 7: "저장하기" CTA 버튼 (파란 필)
- Tier 8: 상단 네비게이션 바 + 푸터 (배경)

## 5. Interaction Elements
| Element | Type | Likely Trigger |
|---|---|---|
| "저장하기" 버튼 | button | 북마크 생성 API 호출 |
| 탭 (추천이유/생활물가/항공권/현지뉴스) | tab | 탭 콘텐츠 전환 |
| "더보기 바로가기" 링크 | link | 뉴스 전체 목록 페이지 이동 |
| 기사 카드 "기사 보기" 링크 | link | 외부 뉴스 기사 URL 오픈 |
| 모달 닫기 | button | 모달 닫기 |

## 6. Data Display Patterns
| Area | Display Type | Dynamic Data |
|---|---|---|
| 뉴스 요약 불릿 | text-list | Yes (GET /api/news/{countryId}) |
| GOOD NEWS 배지 | status indicator | Yes |
| 기사 카드 목록 | card-list | Yes |
| 기사 썸네일 | image | Yes |

## 7. Repeated Patterns
뉴스 요약 항목: 볼드 카테고리 레이블 + 내용 텍스트 구조 3회 반복. 기사 카드: 썸네일 + 제목 + 출처/날짜 + "기사 보기" 링크 구조 3회 반복.

## 8. Dominant Element & Information Density
- **Dominant**: "시드니" 목적지 헤딩 — 최대 폰트 + 흰색 고대비 + 드롭 쉐도우 + 고채도 사진 배경
- **Density**: Medium — 우측에 뉴스 요약 + 기사 목록이 나란히 배치되나 섹션 내 여백 유지
