# UI Structure Analysis: 국가별 상세 물가페이지

## 1. Layout Pattern
단일 중앙 정렬 컨테이너 내 수직 스택. 전체 너비 헤더 + KPI 카드 행(2열) + 물가 비교 상세 섹션(비대칭 2열) + 주요 도시 물가 카드 그리드(4열) + 푸터. 전체 페이지 레이아웃.

## 2. Estimated Grid Structure
12컬럼 기반 변형 추정. 상단 KPI 카드 행: 2열 동등 분할(약 50:50). 물가 비교 상세 섹션: 좌 고정 국가 선택 패널(약 35%) + 우 콘텐츠 영역(약 65%) 비대칭 2열. 주요 도시 물가: 4열 균등 카드 그리드. 섹션 간 수평 구분선으로 섹션 경계 명시. 그리드 이탈 없음.

## 3. Visual Hierarchy Order
- Tier 1: 좌상단 국가명 "일본" 대형 타이포그래피 — 페이지 최상단, 최대 폰트, 고대비
- Tier 2: 좌측 KPI 카드 수치 "912.45" — 대형 숫자, 굵은 웨이트
- Tier 3: 우측 KPI 카드 수치 "0.84" + 도넛 차트
- Tier 4: "물가 비교 상세" 섹션 헤더 + 내부 테이블형 콘텐츠
- Tier 5: 좌측 라인 차트 (저채도, 시각 가중치 낮음)
- Tier 6: "주요 도시 물가" 섹션 헤더 + 4개 도시 카드
- Tier 7: 카드 내부 도시명, 가격 텍스트, 사진 썸네일
- Tier 8: 상단 네비게이션 바
- Tier 9: 하단 푸터

## 7. Repeated Patterns
도시 카드: 이미지 + 도시명 + 물가 설명 텍스트 구조 4회 반복 (4열 그리드). 물가 항목 비교 테이블 행: 카테고리 레이블 + 금액 수치 패턴 반복.

## 8. Dominant Element & Information Density
- **Dominant**: 국가명 "일본" 텍스트 — 최대 폰트 크기 + 좌상단 앵커 포지션 + 고대비 색상
- **Density**: Medium 전체 / 물가 비교 상세 섹션 내부는 High(텍스트 항목 밀집) / 도시 카드 섹션은 Low~Medium
