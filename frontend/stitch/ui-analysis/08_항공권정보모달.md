# UI Structure Analysis: 항공권 정보 모달

## 1. Layout Pattern
modal-overlay. 배경 페이지 위 중앙 모달. 모달 내부: 비대칭 2열 (좌: 목적지 카드 약 30–35% / 우: 탭+콘텐츠 약 65–70%). 활성 탭: "항공권 정보". 우측 콘텐츠: 날짜 탭 행 + 가격 트렌드 라인 차트.

## 2. Estimated Grid Structure
모달 내부: 명시적 12컬럼 없음. 좌우 비대칭 2-zone 분할. 우측 내부: 단일 컬럼 수직 스택 (탭 바 → 날짜 탭 행 → 섹션 헤더 → 라인 차트). 날짜 탭은 전체 우측 컬럼 너비 수평 배열.

## 3. Visual Hierarchy Order
- Tier 1: "시드니" 목적지 헤딩 — 좌측 카드 하단, 최대 폰트, 흰색 고대비 (사진 배경 위)
- Tier 2: "Price Trends" 섹션 헤딩 — 우측 컬럼 최대 텍스트
- Tier 3: 활성 날짜 탭 "2024.06" — 어두운 필 배경 차별화
- Tier 4: 라인 차트 + "432,000원" 데이터 포인트 콜아웃
- Tier 5: 차트 축 레이블, 서브 레이블
- Tier 6: 상단 네비게이션 바
- Tier 7: "저장하기" CTA 버튼 (파란 필)
- Tier 8: 푸터

## 5. Interaction Elements
| Element | Type | Likely Trigger |
|---|---|---|
| "저장하기" 버튼 | button | 북마크 생성 API 호출 |
| 탭 (추천이유/생활물가/항공권/현지뉴스) | tab | 탭 콘텐츠 전환 |
| 날짜 탭 (2024.05~2024.10) | tab | 선택 월 가격 데이터 조회 |
| 1 Monthly / 6 Monthly 토글 | toggle | 차트 기간 전환 |
| 모달 닫기 | button | 모달 닫기 |

## 6. Data Display Patterns
| Area | Display Type | Dynamic Data |
|---|---|---|
| 가격 트렌드 라인 차트 | chart (line) | Yes |
| 날짜별 최저가 콜아웃 | stat-number | Yes |
| 날짜 탭 리스트 | badge/tag | Yes |

## 7. Repeated Patterns
날짜 탭: 동일 스타일 탭 아이템 6–7개 반복 (수평). 탭 아이템 (도시 상세 모달 공통): 4회 반복.

## 8. Dominant Element & Information Density
- **Dominant**: "시드니" 목적지 헤딩 — 최대 폰트 + 흰색 고대비 + 사진 배경 그라디언트 위
- **Density**: Medium — 우측에 탭+차트 집중되나 차트가 넓은 영역을 단일 시각화로 점유해 여백 확보됨
